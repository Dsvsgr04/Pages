3. SecureVision (AI Based Exam Proctoring System)

Project Details

Final year B.Tech project — an AI-powered real-time exam proctoring system built to detect student cheating behaviour using webcam, audio, and face analysis.

Features:

  • Phone detection using a TensorFlow Lite model
  • Face and mouth detection using MediaPipe
  • Speech transcription using Google Speech Recognition
  • Auto-logging of cheating events with timestamp and confidence score to CSV

Technologies Used: Python, TensorFlow, Keras, OpenCV, MediaPipe, SpeechRecognition, PyPDF2

Tools: TensorFlow Lite, Git, GitHub, VS Code, PyAudio

Challenges: Combining multiple detection models (visual, audio, and text) into one real-time pipeline with reliable accuracy.

Future Enhancements: Improve detection accuracy in low-light conditions and add a dashboard for reviewing flagged events.
